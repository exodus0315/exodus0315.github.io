HTML input 태그 활용 (엔터 눌렀을 때 이벤트 발생 등)

March 9, 2022

input 태그란 사용자에게 입력(데이터, 값)을 받을 수 있는 태그입니다.

사용자의 데이터를 받을 수 있기 떄문에 정말 많이 쓰이고 다양하게 활용이 가능한 태그입니다.

프로젝트 연습을 하다가 HTML input tag를 자주 써봤고, 앞으로도 자주 쓸 것 같아서 몇가지를 기록해두고자 합니다.

<input type="text" />

input 태그의 기본 타입(type)은 text이며, 타입(type) 특성에 따라 다양하게 응용이 가능합니다.

타입은 text 뿐 아니라 button, file, password, tel 등 정말 다양합니다.

타입에 대해 자세히 알고 싶으시면 MDN input 유형 에서 비교하며 보실 수 있습니다.

저는 타입에 대해 좀 더 분석하며 비교하기보다는 제가 썼던, 앞으로도 쓸 것 같은 몇가지만 정리하려고 합니다.

input 태그 안에 문구(힌트) 보이게 하기#

placeholder 속성을 사용하시면 해당 input 태그가 어떤 용도, 어떻게 쓸 수 있는지 힌트를 줄 수 있습니다.

<input type="search" placeholder="Search name" />

위와 같이 사용하면 input 태그 안에 Search name 이라는 문구가 기록되어

사용자는 “이름을 입력하면 되겠구나.” 라고 자연스럽게 생각할 수 있습니다.

참고로 search 타입을 사용하면 input 우측에 [ X ]가 생겨서 내용을 쉽게 지울 수 있습니다.

input 태그 선택 시 안에 있는 내용 지우기#

input 객체를 다시 선택하면 입력했던 내용들을 초기화시키는 방법입니다.

아래와 같이 onfocus 속성을 쓰시면 됩니다.

onfocus="this.value=''"

위처럼 사용하시면 input 선택 시, 선택한 input의 값이 비워지게 됩니다.

onfocus와 반대로 onblur를 사용하시면 해당 input를 선택하지 않았을 때의 기능도 구현할 수 있습니다.

<input
  type="text"
  placeholder="Search name"
  onfocus="this.value=''" 
  onblur="inputBlur()"
/>

위와 같이 자바스크립트(Javascript)와 함께 사용도 가능합니다.

특정 키 입력 시 자바스크립트 함수 실행시키기#

저는 input에서 검색값을 입력한 뒤 키보드의 엔터키를 눌러 즉시 검색이 되도록 할 때 사용했습니다.

onKeypress="javascript:if(event.keyCode==13){search()}"

13번 키코드(keyCode)를 눌렀을 때 자바스크립트의 search 함수가 실행되도록 onKeypress를 사용했습니다.

참고로 13번 키 코드는 키보드의 엔터(Enter) 입니다.

search() 와 같이 자바스크립트 함수는 미리 정의해주셔야 합니다.

MDN keyCode 에 들어가시면 더 다양한 키코드를 보실 수 있습니다.

오늘 정리했던 모든 것들을 사용해보면

<input
  type="text"
  placeholder="Search name"
  onfocus="this.value=''"
  onKeypress="javascript:if(event.keyCode==13){search()}"
/>

위와 같이 사용할 수 있습니다.
